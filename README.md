# 在线考试系统

基于 FastAPI 的在线考试系统，支持 MySQL、Python、Linux Shell 题库。

## 快速开始

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 启动服务
```bash
python main.py
```

### 3. 访问系统
打开浏览器访问: http://localhost:8000

## 默认账号
- 管理员: admin / admin123
- 教师: teacher / teacher123
- 学生: student / student123

## 功能特性
- 用户管理（注册/登录）
- 题库管理（CRUD）
- 在线答题
- 自动评分
- 考试历史
